# Путешествие по России

Одностраничный сайт, который посвящен необычным местам по всей России. В проекте есть коллаж фотографий, карточки с их описанием.

Проект направлен на отработку навыков адаптивной верстки (в т.ч. flex, grid)\
С использованием стандартных технологий в виде HTML и CSS. Присутствует семантическая разметка и БЭМ-методология.
